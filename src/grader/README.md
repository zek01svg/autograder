# ☕ AutoGrader Core (Java)

The backend execution engine that handles student code validation and Docker-based testing.

## 🚀 CLI Usage

```sh
# Windows
scripts\run.bat --submissions <zips-path>

# macOS/Linux
./scripts/run.sh --submissions <zips-path>
```

## 🏗 Core Modules
- **`core/`**: Pipeline orchestration and Docker execution.
- **`validator/`**: File structure and security validation.
- **`report/`**: HTML/CSV output generation.
